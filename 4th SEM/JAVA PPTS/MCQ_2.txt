1. What will be the output of the following code?

class Test {
    private int x;

    Test() {
        this.x = 10;
    }

    Test(Test obj) {
        this.x = obj.x + 10;
    }

    void display() {
        System.out.println("x = " + x);
    }

    public static void main(String[] args) {
        Test t1 = new Test();
        Test t2 = new Test(t1);
        t2.display();
    }
}
A) x = 10
B) x = 20
C) Compilation Error
D) Runtime Error

✅ Answer: B) x = 20

🔹 Explanation: The copy constructor Test(Test obj) initializes x as obj.x + 10, where obj.x = 10. So, t2.x = 10 + 10 = 20.
